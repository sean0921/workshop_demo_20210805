# Demo for JME Workshop in NCKU 2021

這個簡單的 demo 筆記本主要呈現如何結合 Jupyter Widget (`ipywidget`) 和 PyGMT 達成簡單的互動式繪圖效果，有別於以往我們接觸 GMT 只能藉由「刻腳本」(撰寫 Shell Script)來達成，這樣的方法給予我們更直覺的方式，利用 GMT 這個強大的繪圖工具的可能。
